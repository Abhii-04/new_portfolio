# Portfolio - Abhishek Yadav

A personal portfolio website showcasing my experience, projects, and skills as a FullStack Developer.

## Tech Stack

- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Hosting**: Vercel

## Project Structure

```
├── assets/          # Images and static assets
├── css/             # Stylesheets
├── js/              # JavaScript files
├── index.html       # Main portfolio page
├── naiyo24.html     # Internship details page
└── vercel.json      # Vercel configuration
```

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000`

## Deployment

This project is configured for deployment on Vercel. Simply:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy!

The `vercel.json` file handles routing for clean URLs.
